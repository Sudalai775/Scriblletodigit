# ✍️ Scribble Digital

### Handwritten Notes → Clean, Structured Digital Text

Scribble Digital is an AI-powered OCR application that converts handwritten notes into clean, editable, and structured digital text. The platform combines EasyOCR, OpenCV image enhancement, and optional AI correction to improve extraction accuracy and readability.

---
![Website Screenshot](https://github.com/Sudalai775/Scriblletodigit/blob/main/Website.png?raw=true)
## 🚀 Features

* 📸 Upload handwritten note images
* 🔍 OCR text extraction using EasyOCR
* 🎨 OpenCV image preprocessing
* 🤖 AI-powered text correction (DeepSeek API)
* 🧹 Local text cleaning pipeline
* 📝 Structured note generation
* 🔄 Change highlighting
* 🩺 OCR quality diagnostics
* 📜 Session history tracking
* 📦 Export as TXT, Markdown, and JSON
* 🌙 Simple and responsive interface
